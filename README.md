## 功能简介：

* Shiro验证---后台登录
* 安全退出后台,修改用户密码---MD5保存密码
* 修改用户信息----上传图片---个人头像
* 文章管理--文章CRUD---分别使用UE和KE编辑器
* 图片管理--图片CRUD--JQuery图片上传插件Uploadify

## 框架实现：

EasyUI+Spring+SpringMVC+MyBatis
